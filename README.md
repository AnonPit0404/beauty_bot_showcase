# 💅 Beauty Bot for Tatyana
### Telegram-бот для онлайн-записи к мастеру красоты

Клиент выбирает дату через интерактивный календарь, оставляет имя и телефон — мастер получает готовую запись. Без сторонних сервисов, всё в Telegram.

## ⚡ Возможности

### 👤 Для клиента


| Функция | Описание |
| :--- | :--- |
| **📅 Запись** | Интерактивный календарь — текущий и следующий месяц |
| **✅ Валидация** | Проверка телефона по коду страны и количеству цифр |
| **🔔 Напоминание** | Автоматически за 24 часа до визита |
| **👁 Управление** | Просмотр и отмена своей записи |
| **💰 Прайс** | Актуальные цены прямо в боте |
| **🖼 Портфолио** | Галерея работ мастера |

### 🔧 Для мастера (панель администратора)
* **Добавление слотов по диапазону и шагу** — например `10:00–18:00` каждые 30 или 60 мин.
* **Управление расписанием** — удаление слотов и закрытие дня одной кнопкой.
* **Просмотр** — детальное расписание на любую выбранную дату.

---

## 🛡 Надёжность
* **Защита от двойной записи** на одно и то же время (Race Condition).
* **Автоматическое освобождение** «зависших» броней, если клиент не завершил ввод данных (`pending` > 15 мин).
* **Восстановление всех напоминаний** из базы данных после перезапуска бота.
* **Логирование ошибок** в реальном времени в отдельный Telegram-канал для мастера.

## 🗂 Стек технологий


| Компонент | Технология |
| :--- | :--- |
| **Язык** | Python 3.10+ |

| **База данных** | SQLite через aiosqlite (асинхронно) |
| **Фоновые задачи** | APScheduler — AsyncIOScheduler |

## 📁 Структура проекта

```text
beauty_bot/
├── bot.py              # Точка входа, запуск поллинга и шедулера
├── config.py           # Загрузка и валидация переменных окружения
├── states.py           # FSM-состояния для сценариев aiogram
├── database.py         # Асинхронный класс работы с SQLite (aiosqlite)
├── keyboards.py        # Генераторы inline-кнопок и календаря
├── handlers/
│   ├── client.py       # Обработка команд и кнопок клиента
│   └── admin.py        # Логика панели управления мастера
└── utils/
    └── scheduler.py    # Настройка APScheduler и тасок напоминаний
```
